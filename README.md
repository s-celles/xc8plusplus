# xc8plusplus

C++ to C transpiler for Microchip's XC8 compiler. Enables C++ development for 8-bit PIC microcontrollers.

## Quick Start

```bash
# Transpile C++ to C
python src/xc8_transpiler.py input.cpp output.c

# Compile with XC8  
xc8 output.c
```
